x,yの2次式

    xy+◯x+□y+△=0 → (x+…)(y+…)=k と変形
  • → k≠0 ならば 双曲線
  • → k=0 ならば 2直線の和集合
    ◯x2+□y2+(x,yの1次式)=0 → a(x+…)2+b(y+…)2=k と変形
      → a=b のとき
    • → kがa,bと同符号 ならば 円
    • → k=0 ならば 点
    • → kがa,bと異符号 ならば 図形なし（空集合）
      → a≠b かつ a,bが同符号 のとき
    • → kがa,bと同符号 ならば 楕円
    • → k=0 ならば 点
    • → kがa,bと異符号 ならば 図形なし（空集合）
      → a,b が異符号のとき → (x,yの1次式)(x,yの1次式)=k と変形
